## What I'm building → Vigilant AI

> **A fraud detection system that protects ordinary Africans from 
> mobile money scams — using AI, plain language explanations, 
> and support for Swahili and English.**

Most fraud detection tools were built for Western banking systems.
They don't understand M-Pesa flows, Swahili scam patterns, 
SIM swap fraud, or the reality of low-bandwidth African networks.

Vigilant AI is built specifically for this gap.

**What it does:**
- Scans SMS messages and flags M-Pesa scams in real time
- Explains *why* something is suspicious in simple language
- Works in both English and Swahili
- Built to run on low-bandwidth, low-spec devices

**Current status:** 🔨 Building Phase 1 — Rule Engine + Web App

**Where the project lives:** [`vigilant-ai`](https://github.com/charlesKariuki-001/vigilant-ai)
