* 🏗️ **Systems Architecture** — Scalable, offline-first, and containerized (Docker) cloud solutions.

## 🚀 Flagship Project: [Vigilant AI](https://github.com/vigilantAIAfrica)

**A real-time, AI-powered fraud detection engine for the African mobile money ecosystem.**

* **Core Logic:** FastAPI & Python-based detection of M-Pesa scams and phishing.
* **Innovation:** Swahili-localized NLP for "last-mile" user protection.
* **Infrastructure:** Containerized deployment (Docker) with integrated network monitoring (Snort).
